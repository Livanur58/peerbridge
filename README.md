# PeerBridge
## Proje Adı
PeerBridge: Akran Akademik Destek ve Psikolojik Danışman Entegrasyonlu Öğrenme Platformu
## Projenin Amacı
Bu projenin amacı, üniversite öğrencilerinin akademik başarılarını artırmak, akran öğrenmesini desteklemek ve psikolojik danışmanlara yazılı olarak erişimi kolaylaştırmaktır.
## Proje Kapsamı
PeerBridge platformu aşağıdaki modülleri içermektedir:
- Ders notu paylaşımı
- Akran eşleştirme sistemi
- Grup çalışma alanları
- Psikolojik danışman mesajlaşma modülü
## Yöntem
Proje; ihtiyaç analizi, yazılım tasarımı ve pilot uygulama aşamalarından oluşmaktadır.
